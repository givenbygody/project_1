# Korean Food Landing Page

## Project Overview
This project is a **single-page web application** built with **Bootstrap**, inspired by the vibrant atmosphere of **Korean street food culture**.

The main idea of the project is to create a **creative, modern, and user-friendly menu page** that presents Korean food in an attractive and engaging way.

The website combines:
- modern responsive design,
- creative visual presentation of menu items,
- user-friendly navigation,
- adaptive layout for different screen sizes.

---

## Project Goals
The primary goals of this project are:

1. **Create an attractive one-page website**
   - Build a visually appealing landing page dedicated to Korean food.
   - Present menu items in a stylish and structured format.

2. **Optimize user experience**
   - Make the interface intuitive and easy to navigate.
   - Ensure responsiveness across desktops, tablets, and mobile devices.

3. **Track website development**
   - Use the project as a way to document and monitor the website’s growth.
   - Gradually improve the design and structure.

4. **Implement new features**
   - Continuously add new functionalities and creative elements.
   - Experiment with UI improvements, animations, and user-oriented enhancements.

---

## Technologies Used
This project is built with:

- **HTML5** — page structure
- **CSS3** — custom styling
- **Bootstrap 5** — responsive layout and components
- **JavaScript** — interactive functionality
